Compute Text Similarity

What is this?

This website utilizes the Universal Sentence Encoder to calculate how similar a bunch of strings are to each other. What's cool about this sentence encoder, is that it computes semantic similarity, meaning how close pieces of text are to each other in their meaning, and not in the words they share.

How does it work?

Type or paste any paragraphs or sentences you'd like into the 2 fields. If you would like to compute the similarity between more strings, you can do that as well. Upon computation, a heatmap will be generated showing how each text is similar to every other text. Try it out and have fun!
